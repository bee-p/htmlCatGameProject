## 🐣 게임 소개 🎈

### 🛠 제작 기간 및 규모
제작 날짜: `2020년도 2학기` </br>
소요 기간: `일주일 이내` </br>
개발 규모: `1인`

### ✏️ 기획 의도
웹페이지 기말 프로젝트를 구상하던 도중, </br>
무언가 스크롤바에 매달려 있다면 귀여울 것 같아 </br>
이를 간단한 게임으로 표현해보았습니다.

### 🕹️ 화면 설명
<img src="https://github.com/bee-p/htmlCatGameProject/assets/68233618/c611a650-cad5-40d3-a6d3-2d15be84167d" width="70%">

> 기본 화면입니다. </br>
> 레트로한 `화면 모니터`처럼 보이도록 웹페이지를 구성했습니다. </br>
> 오른쪽 하단에서 현재 시간과 날짜를 확인할 수 있습니다. </br>
> 동작할 수 있는 버튼은 `CAT`(게임 실행), `도움말` 버튼입니다. </br>
> 웹페이지를 구성하는 모든 이미지 리소스들은 직접 그렸습니다.

</br>

<img src="https://github.com/bee-p/htmlCatGameProject/assets/68233618/5db09260-04cc-4c8c-971d-81653a968aeb" width="70%">

> 도움말 버튼을 누르면 CAT 게임의 `조작 방법`, `규칙 설명`이 나옵니다. </br>
> 털실을 좀 더 잘 피할 수 있는 방법도 적어두었습니다. </br>
> 오른쪽 위 `X` 버튼을 통해 창을 닫을 수 있습니다.

</br>

<img src="https://github.com/bee-p/htmlCatGameProject/assets/68233618/ed5047c3-c0ee-4925-bd49-b60c7e5d5f97" width="70%">

> CAT 버튼을 눌렀을 때 실행되는 `게임 화면`입니다. </br>
> 마치 게임 실행 파일을 누르는 것처럼 보이게 하기 위해 `창 모드`의 형태로 나타나게 하였습니다. </br>
> `방향키`로 조작하며 날아오는 털실을 피할 수 있습니다. </br>
> 오른쪽 위 `X` 버튼을 통해 창을 닫을 수 있습니다.

</br>

<img src="https://github.com/bee-p/htmlCatGameProject/assets/68233618/f5bb0754-fdef-4f7c-89d0-061a0cc116aa" width="70%">

> 게임 진행 도중, 고양이가 털실에 닿으면 나오는 `Game Over` 창입니다. </br>
> 달성한 `점수`를 확인할 수 있습니다.
